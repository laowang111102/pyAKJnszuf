# 学生宿舍信息管理系统 java1079

## 项目概述

学生宿舍信息管理系统是基于前后端分离架构开发，适用于学校宿舍管理的软件。系统提供完整的功能模块，满足不同角色的使用需求，包括系统管理员、宿舍管理员以及学生。

## 技术栈

- 开发环境：IDEA、JDK 1.8、Maven、Node.js
- 后端技术：Java、SpringBoot、MyBatis-Plus
- 前端技术：Vue
- 数据库：MySQL

## 功能模块

### 系统角色

#### 系统管理员模块
- 学生信息管理
- 宿管信息管理
- 楼宇信息管理
- 房间信息管理
- 公告信息管理
- 报修信息管理
- 调宿信息管理
- 访客管理
- 个人信息

#### 宿舍管理员模块
- 学生信息管理
- 楼宇信息管理
- 房间信息管理
- 公告信息管理
- 报修信息管理
- 调宿信息管理
- 访客管理
- 个人信息

#### 学生模块
- 学生首页
- 查看公告
- 我的宿舍
- 申请调宿
- 报修申请
- 个人信息

## 运行环境

- 后端：Java 1.8及以上版本环境
- 前端：现代浏览器
- 数据库：MySQL 5.7及以上版本

- 前后端分离的架构设计，满足现代软件开发需求，提高开发效率与系统性能。
